# SAMVIL v3.15 Phase 13 — Release Check Runner

**Date:** 2026-04-26
**Base:** v3.14.0 Release Readiness Gate
**Target:** v3.15.0

## Goal

Make release readiness evidence executable. Phase 13 adds a deterministic
runner that executes release check commands, captures exit code/duration/output
summaries, writes `.samvil/release-report.json`, and feeds the existing release
gate without manual check JSON.

## Phase 13 Outputs

| Area | Output |
|---|---|
| Runner | execute named release check commands |
| Evidence | exit code, duration, stdout/stderr snippets |
| CLI | `scripts/run-release-checks.py` |
| MCP | release check runner tool |
| Report | auto-generated `.samvil/release-report.json` |
| Dogfood script | `scripts/phase13-release-check-runner-dogfood.py` |
| Release | `v3.15.0` |

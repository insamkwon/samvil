# SAMVIL v3.25 Phase 23 — Evolve Intake Context

## Goal

Phase 22 routes blocked QA to a next skill. Phase 23 gives `samvil-evolve`
a file-based intake context so it can start from the exact seed, QA issue IDs,
convergence reason, routing decision, and ground-truth artifacts without
depending on conversation history or session DB state.

## Phase 23 Outputs

| Area | Output |
| --- | --- |
| Core | file-based evolve context builder in `evolve_loop.py` |
| Artifact | `.samvil/evolve-context.json` |
| MCP | `build_evolve_context`, `materialize_evolve_context` |
| Runtime | run-report and `samvil-status` evolve context summaries |
| Dogfood | `scripts/phase23-evolve-intake-context-dogfood.py` |
| Tests | evolve context, status, telemetry, MCP smoke, Phase 23 dogfood |

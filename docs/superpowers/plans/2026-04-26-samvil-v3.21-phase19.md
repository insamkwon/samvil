# SAMVIL v3.21 Phase 19 — QA Synthesis Gate

**Date:** 2026-04-26
**Base:** v3.20.0 Independent Evidence Contract
**Target:** v3.21.0

## Goal

Make independent QA evidence centrally decidable. Phase 18 kept the principle
from drifting; Phase 19 adds a machine-readable synthesis gate that turns Pass
1, independent Pass 2, and independent Pass 3 evidence into one deterministic
`PASS` / `REVISE` / `FAIL` decision owned by the main session.

## Phase 19 Outputs

| Area | Output |
|---|---|
| Synthesis core | `mcp/samvil_mcp/qa_synthesis.py` |
| MCP tool | `synthesize_qa_evidence` |
| Dogfood | `scripts/phase19-qa-synthesis-gate-dogfood.py` |
| Tests | `mcp/tests/test_qa_synthesis.py`, `mcp/tests/test_phase19_qa_synthesis_gate_dogfood.py` |
| Release runner | Phase 19 dogfood required before Phase 18 and pre-commit |
| Release | `v3.21.0` |

## Workstreams

- [x] Count `PASS`, `PARTIAL`, `UNIMPLEMENTED`, and `FAIL` functional outcomes.

## Non-Goals

- Parsing arbitrary markdown QA reports.
- Letting independent agents write state or events directly.
- Replacing runtime Playwright evidence collection.
- Changing minimal tier inline QA behavior.

## Release Bar

- Phase 19 dogfood proves central synthesis handles pass, revise, fail, quality,
  and protected-write scenarios.
- Default release runner includes Phase 19 and passes all checks.
- Live publisher flow pushes the tag only after remote artifact gate passes.

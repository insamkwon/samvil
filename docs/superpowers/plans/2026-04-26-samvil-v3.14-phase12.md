# SAMVIL v3.14 Phase 12 — Release Readiness Gate

**Date:** 2026-04-26
**Base:** v3.13.0 Repair Orchestration Gate
**Target:** v3.14.0

## Goal

Make final release readiness deterministic. Phase 12 records release check
evidence, evaluates a release gate, and surfaces the result in run reports and
status so SAMVIL can distinguish "repair verified" from "safe to release".

## Phase 12 Outputs

| Area | Output |
|---|---|
| Release report | `.samvil/release-report.json` |
| Release gate | deterministic pass/blocked verdict |
| Repair dependency | release blocks when repair gate is blocked |
| Run report | release summary + release gate |
| Status surface | release gate verdict and next action |
| Dogfood script | `scripts/phase12-release-readiness-dogfood.py` |
| Release | `v3.14.0` |

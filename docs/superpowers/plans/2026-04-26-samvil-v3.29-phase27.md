# SAMVIL v3.29 Phase 27 — Rebuild Reentry Contract

## Goal

Phase 26 writes a portable rebuild handoff. Phase 27 materializes the exact
scaffold reentry input so the next host can restart from the evolved seed
without inferring seed path, version, or hash from conversation context.

| MCP | `build_rebuild_reentry`, `materialize_rebuild_reentry` |

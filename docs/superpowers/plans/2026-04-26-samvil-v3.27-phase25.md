# SAMVIL v3.27 Phase 25 — Evolve Apply Plan

## Goal

Phase 24 produces reviewable evolve proposals. Phase 25 turns those proposals
into guarded seed patch previews and applies them only when the current seed
hash still matches the plan.

## Phase 25 Outputs

| Area | Output |
| --- | --- |
| Core | `mcp/samvil_mcp/evolve_apply.py` |
| Artifacts | `.samvil/evolve-apply-plan.json`, `.samvil/evolved-seed.preview.json`, `.samvil/evolve-apply-report.md` |
| Apply | guarded `project.seed.json` update plus `seed_history/vN.json` backup |
| MCP | `build_evolve_apply_plan`, `materialize_evolve_apply_plan`, `apply_evolve_apply_plan` |
| Runtime | run-report and `samvil-status` apply summaries |
| Dogfood | `scripts/phase25-evolve-apply-dogfood.py` |
| Tests | apply plan, status, telemetry, MCP smoke, Phase 25 dogfood |

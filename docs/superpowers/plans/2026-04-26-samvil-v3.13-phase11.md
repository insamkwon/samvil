# SAMVIL v3.13 Phase 11 — Repair Orchestration Gate

**Date:** 2026-04-26
**Base:** v3.12.0 Inspection Repair Execution Loop
**Target:** v3.13.0

## Goal

Make repair state part of SAMVIL's deterministic progression decision. Phase 11
adds a repair gate verdict, integrates repair summary into run reports/status,
and records repair lifecycle events so QA/complete can be blocked until repair
is verified.

## Phase 11 Outputs

| Area | Output |
|---|---|
| Repair gate | deterministic pass/blocked/not-applicable verdict |
| Run report | repair summary + repair gate |
| Status surface | repair gate verdict and next action |
| Stage events | repair_started, repair_plan_generated, repair_applied, repair_verified, repair_failed |
| Dogfood script | `scripts/phase11-repair-orchestration-dogfood.py` |
| Release | `v3.13.0` |

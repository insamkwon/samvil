# SAMVIL v3.9 Phase 7 — Browser Runtime Dogfood

**Date:** 2026-04-26
**Base:** v3.8.0 Real Runtime Dogfood
**Target:** v3.9.0

## Goal

Prove two generated apps run in a real browser after real package install:

- Vite React SaaS dashboard: DOM content and filter interaction.
- Vite Phaser browser game: canvas nonblank pixel, keyboard input, score, and
  restart interaction.

This phase intentionally allows network-dependent `npm install` because that is
the point of the dogfood. Failures are treated as real environment/package
evidence, not ignored noise.

# SAMVIL v3.16 Phase 14 — Release Evidence Bundle

**Date:** 2026-04-26
**Base:** v3.15.0 Release Check Runner
**Target:** v3.16.0

## Goal

Make release evidence reviewable in one file. Phase 14 turns the runner-generated
`.samvil/release-report.json` into a human-readable `.samvil/release-summary.md`
that captures the release gate verdict, git identity, version sync state, check
exit codes/durations, and failure output tails.

## Phase 14 Outputs

| Area | Output |
|---|---|
| Bundle | `.samvil/release-summary.md` |
| Bundle metadata | git branch/head/tags and version sync state |
| CLI | `scripts/build-release-bundle.py` |
| MCP | release evidence bundle tools |
| Status surface | latest release bundle path |
| Dogfood script | `scripts/phase14-release-evidence-bundle-dogfood.py` |
| Release | `v3.16.0` |

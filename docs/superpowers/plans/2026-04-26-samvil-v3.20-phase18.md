# SAMVIL v3.20 Phase 18 — Independent Evidence Contract

**Date:** 2026-04-26
**Base:** v3.19.1 Verified Release Publisher
**Target:** v3.20.0

## Goal

Lock the Independent Evidence, Central Verdict principle into an executable
contract. The design/build/QA/evolve text already contains the core behavior;
Phase 18 adds a dogfood validator and promotes it into the release runner so
future edits cannot silently drift from the intended pipeline.

## Non-Goals

- Rewriting the full QA flow.
- Changing minimal tier QA behavior.
- Making independent agents write state, reports, or events.
- Adding external storage for QA evidence.

## Release Bar

- Phase 18 dogfood proves the design/build/QA/evolve contract is present.
- Default release runner includes Phase 18 and passes all checks.
- Live publisher flow pushes the tag only after remote artifact gate passes.

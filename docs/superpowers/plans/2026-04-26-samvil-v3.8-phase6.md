# SAMVIL v3.8 Phase 6 — Real Runtime Dogfood

**Date:** 2026-04-26
**Base:** v3.7.0 Dual Full-Chain Dogfood
**Target:** v3.8.0

## Goal

Move beyond file-only dogfood by proving generated apps can pass a local build
command, start a real HTTP runtime, and serve user-visible content that matches
the domain-specific expectations.

The runtime dogfood remains deterministic and network-free: no package install,
no external APIs, no browser automation dependency. It still exercises actual
`npm run build`, `npm start`, local port binding, HTTP health checks, and HTML
response validation.

# SAMVIL v3.18 Phase 16 — Remote Release Gate

**Date:** 2026-04-26
**Base:** v3.17.3 External CI Mirror
**Target:** v3.18.0

## Goal

Make remote CI evidence a deterministic release gate. Phase 16 verifies that
the GitHub Actions run for the current HEAD completed successfully and that the
uploaded `samvil-release-evidence` artifact contains a passing
`release-runner.json` gate before a release is considered ready.

## Phase 16 Outputs

| Area | Output |
|---|---|
| Gate core | `mcp/samvil_mcp/remote_release.py` |
| CLI | `scripts/check-remote-release-gate.py` |
| Evidence | GitHub Actions run metadata + `release-runner.json` |
| Fixtures | pass/fail remote run and runner artifact fixtures |
| Tests | unit + CLI coverage for pass, failed run, blocked artifact, head mismatch |
| Release | `v3.18.0` |

## Workstreams

### W5 — Release

## Non-Goals

- Replacing GitHub branch protection.
- Adding package publishing.
- Running remote CI as part of local pre-commit.

## Release Bar

- Remote run metadata and artifact gate must both pass.
- Current HEAD mismatch blocks release.
- Fixture tests capture pass and failure modes without requiring GitHub access.
- Live `gh` check succeeds for the latest main release run before tagging.

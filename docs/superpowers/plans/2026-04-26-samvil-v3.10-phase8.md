# SAMVIL v3.10 Phase 8 — Real App Inspection Gate

**Date:** 2026-04-26
**Base:** v3.9.1 Browser Runtime Dogfood + telemetry patch
**Target:** v3.10.0

## Goal

Move from "the generated app runs in a browser" to "the generated app passes a
repeatable user-visible inspection gate." Phase 8 records browser evidence in a
project-local inspection report and exposes it through MCP/status surfaces.

## Phase 8 Outputs

| Area | Output |
|---|---|
| Inspection report module | `mcp/samvil_mcp/inspection.py` |
| MCP tools | build/read/render inspection report |
| Browser dogfood script | `scripts/phase8-real-app-inspection.py` |
| Release-bar test | `mcp/tests/test_phase8_real_app_inspection.py` |
| Evidence | console errors, viewport screenshots, DOM/canvas checks, interaction checks, overflow checks |
| Release | `v3.10.0` |

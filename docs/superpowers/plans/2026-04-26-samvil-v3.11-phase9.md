# SAMVIL v3.11 Phase 9 — Inspection Feedback Loop

**Date:** 2026-04-26
**Base:** v3.10.0 Real App Inspection Gate
**Target:** v3.11.0

## Goal

Turn inspection failures into actionable repair loops. Phase 9 classifies failed
inspection checks, attaches repair hints, derives retro observations, and makes
`samvil-status.py` point operators at the first inspection repair action.

## Phase 9 Outputs

| Area | Output |
|---|---|
| Failure taxonomy | console-error, layout-overflow, screenshot-missing, interaction-failed, canvas-blank, viewport-load-failed, evidence-missing |
| Repair hints | report-level failure records with suggested action |
| Retro bridge | inspection failures -> retro observation candidates |
| Status surface | inspection failure next action priority |
| Dogfood script | `scripts/phase9-inspection-feedback-dogfood.py` |
| Release | `v3.11.0` |

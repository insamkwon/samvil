# SAMVIL v3.30 Phase 28 — Post-Rebuild QA Rejudge

## Goal

Phase 27 writes exact scaffold reentry input. Phase 28 closes the next gap:
after scaffold/rebuild output exists, SAMVIL must route back to QA with a
machine-readable request that pins seed path, version, hash, previous QA
issues, and rebuilt artifacts.

# SAMVIL v3.7 Phase 5 — Dual Full-Chain Dogfood + Hardening

**Date:** 2026-04-26
**Base:** v3.6.0 Domain Packs
**Target:** v3.7.0

## Goal

Run two contrasting dogfood scenarios through a deterministic full-chain
simulation so SAMVIL can catch regressions across product-domain context,
pattern context, generated project structure, QA checks, telemetry, status, and
retro feedback.

The two scenarios are:

- `saas-dashboard`: validates data, filters, metrics, status surfaces, and
  Domain Pack context for business tools.
- `browser-game`: validates game loop, canvas/input/restart expectations, and
  non-dashboard app behavior.
